Available memory aggregated

  • The purple gradient represents the sum of available memory of all the machines in the group, and it uses the scale in the right side of the graphic
  • The gray line is the average available memory for one machine, and it uses the scale in the left side of the graphic
  • The shadows are minimum and maximum values of the period
  • It helps to check if more memory is needed and if you need to switch between high memory or high cpu instance types
