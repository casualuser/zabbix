zabbix by lorieri

zabbix templates, scripts, and chef automations, mostly for aws deploys

Done:
- cloudwatch template, script and chef automation

Done but not at github yet:

- instructions for multiple aws accounts
- instructions and example of a basic linux template for aws
- varnish template, using trapper items examples for better consistency, speed and distribution
- aggregations by groups, so it is not an issue if an instance dies
   Capaciy view as total resources usage
- auto-register for bootstrapping
- auto-deregister for autoscalling groups (or anything else)
